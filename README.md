# 💰 Expense & Income Tracker

Ceci est une application de gestion de finances personnelles développée avec **Laravel**. Elle permet de suivre les revenus, les dépenses et de calculer le profit mensuel ainsi que l'épargne.

## 🚀 Fonctionnalités
*   **Gestion des Revenus :** Enregistrement des gains.
*   **Suivi des Dépenses :** Visualisation des sorties d'argent par catégorie.
*   **Calcul du Profit :** Vue d'ensemble du reste à vivre à la fin du mois.
*   **Objectifs d'Épargne :** Suivi de la progression des économies.

## 🛠️ Technologies utilisées
*   **Framework :** [Laravel 11](https://laravel.com)
*   **Base de données :** MySQL (via Laragon)
*   **Front-end :** Tailwind CSS / Vite

## ⚙️ Installation
1.  Cloner le projet : `git clone https://github.com/ton-pseudo/ton-projet.git`
2.  Installer les dépendances PHP : `composer install`
3.  Installer les dépendances JS : `npm install`
4.  Configurer le fichier `.env` (base de données)
5.  Lancer les migrations : `php artisan migrate`
6.  Lancer le serveur : `php artisan serve` et `npm run dev`
